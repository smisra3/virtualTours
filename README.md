## virtualTours


# Pre-requisites

1- Must have Node.js and NPM installed on the machine.

# Commands to Run. 

1- git clone https://github.com/smisra3/virtualTours.git

2- cd virtualTours

3- git checkout main

3- npm i

4- npm run start

Open localhost:5500 to view the app working.
